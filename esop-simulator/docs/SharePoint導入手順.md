# SharePoint への格納手順（全社普及用）

最終的な格納はご担当者が行う前提で、迷わず進められるように手順とコツをまとめます。

> **最重要ポイント（先に知っておいてください）**
> 最近のSharePoint Online（モダン環境）は、セキュリティ上の理由で
> **ドキュメントライブラリに置いたHTMLファイルを「その場で実行」できません。**
> `index.html` をそのままアップしてクリックしても、多くの環境では
> ダウンロードされるか、ソースが表示されるだけで、計算ツールとして動きません。
>
> そのため、**「ツール本体はWeb（GitHub Pages）で動かし、SharePointからはそこへ誘導する」**
> 形が、全社普及にもっとも確実でおすすめです。下のおすすめ手順をご覧ください。

---

## おすすめ：SharePointページから公開URLへ誘導（確実・推奨）

### 事前準備
- 「スマホ用URL公開手順.md」の②に沿って **GitHub Pages** を有効化し、公開URLを用意します。
  公開URL例：`https://noguchi-tech.github.io/store-operations-tools/esop-simulator/`

### 手順
1. 全社ポータル等で **新しいページを作成**（または既存ページを編集）。
2. ページに **「埋め込み（Embed）」Webパーツ** を追加。
3. 公開URLを貼り付ける。ページ内にツールがそのまま表示され、その場で計算できます。
   - 埋め込みがブロックされる場合は、テナント管理者に
     **「HTMLフィールドセキュリティ／埋め込み許可ドメイン」に `github.io` を追加**してもらってください。
4. 埋め込みが使えない環境では、代わりに **「リンク」Webパーツ**や見やすい
   **ボタン**を置き、公開URLへ飛ばすだけでも十分です（スマホでもそのまま動きます）。
5. ページを **発行（Publish）** し、全社に周知します。

### この方式の利点
- 利用者は社内ログインのまま、追加の会員登録なしで使える。
- ツールを更新（`main`に反映）すれば、SharePoint側も常に最新になる。
- スマホ・PCどちらでも同じURLで動く。

---

## 代替：HTMLファイルそのものを配布する場合

「Webに出さず、ファイルとして社内に置きたい」場合は次のようにします。
（クリックですぐ起動はしませんが、ダウンロードして開けば動きます）

1. ドキュメントライブラリ（例：全社共有 → 持株会）に `index.html` をアップロード。
   - 分かりやすいよう `持株会シミュレーター.html` にリネームしてもOKです。
2. 利用案内に「ファイルをダウンロードしてから開いてください」と明記。
3. スマホの場合は、ブラウザでダウンロード→開く操作が必要になり、やや手間です。
   全社普及の手軽さでは、上の「公開URLへ誘導」方式に劣ります。

## 格納するファイル

| ファイル | 用途 |
|---|---|
| `esop-simulator/index.html` | ツール本体（これ1つで動作） |
| `esop-simulator/README.md` | ツールの説明 |
| `esop-simulator/口頭説明用スクリプト.md` | 対面説明用のカンペ（説明者用） |
| `esop-simulator/docs/スマホ用URL公開手順.md` | URL公開の手順 |
| 本ファイル | SharePoint格納の手順 |

## 周知文の例（コピペ用）

> 【持株会シミュレーター公開のお知らせ】
> 毎月の積立で将来いくらになるか試算できるツールを公開しました。
> スマホ・PCから、ログイン不要ですぐ使えます。
> ▶ こちら：（公開URLを貼る）
> ※税金・手数料等は考慮しない概算です。投資判断はご自身の責任でお願いします。
