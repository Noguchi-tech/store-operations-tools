# 持株会 評価額シミュレーター

社内向けの持株会シミュレーションツールです。単体の `index.html` だけで動作します（インストール・ネット接続不要）。

## 使い方

- `index.html` をブラウザで開く（スマホ可）。社内配布する場合はこのファイルを共有するだけでOKです。
- 入力を変えると即時に再計算されます。
- 「初期値にリセット」で標準条件に戻せます。
- 「年次表をCSV出力」で年ごとの推移を保存できます。

## できること

- 毎月の拠出額・ボーナス拠出（あり／なし、月、追加額）の設定
- 拠出年数と運用年数を分けて設定（退職後の「運用のみ」期間を再現）
- 年平均成長率（3% / 5% / 7% / 10% / カスタム）の切り替え
- 退職金ポイント口座の信託簿価（固定／一定周期で時価更新）
- 退職後は退職金ポイントを時価評価し「特定口座移管分」として表示
- サマリー、年次推移表、グラフ、3/5/7/10% の比較表

## 計算の前提

- 内部はすべて月次計算（`月次成長率 = (1+年率)^(1/12) − 1`）。
- 野村口座：毎月の拠出額 ÷ その月の株価で株数を積み増し、評価額＝株数 × 株価。
- 退職金ポイント：付与ポイント＝拠出額 × 付与率 ÷ 信託簿価。評価額＝ポイント × 株価。
- 退職（拠出年数の終了）以降は新規拠出なし。退職金ポイントは退職時の時価で評価され、
  以後は特定口座移管分として株価成長に連動して表示します。
- 個人の保有額はデフォルト0。既に保有がある方は「いまの保有状況」に入力してください。

## 注意

税金・配当・手数料・制度変更・株式分割は考慮しない概算です。実際の受取額や課税関係は、
会社制度・証券会社・税務上の扱いにより異なります。投資判断は本人の責任で行ってください。
